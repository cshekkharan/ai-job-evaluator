# AI Job Evaluator

LLM-based evaluation workflow that compares candidate profiles with job descriptions and produces structured outputs.

## Objective
Build a repeatable prompt evaluation system to improve output quality, consistency, and reliability.

## Features
- Prompt-based evaluation
- Match scoring (0–100)
- Summary, strengths, risks
- A/B prompt testing
- Evaluation rubric
- Workflow automation (n8n)

## Workflow
Input → Webhook → LLM → Scoring → Output

## Evaluation Criteria
- Accuracy
- Completeness
- Clarity
- Relevance
- Consistency

## Results
- Tested 75+ prompt variations
- Processed 200+ runs
- Improved consistency by ~30–40%
- Reduced hallucinations by ~25–35%

## Tech Stack
- OpenAI / ChatGPT
- n8n
- REST APIs
- Webhooks
- Google Sheets
